## About

A scrollytelling portfolio site targeting an Information Systems / Business Analyst / Product Management professional direction. Built using progressive scroll reveals, sticky navigation, animated skill bars, and a full-profile content page — following the spec-driven development process from class.

## File Structure

```
index.html      ← Main scrollytelling homepage (6 scroll sections)
about.html      ← Full profile content page (education, experience, skills, leadership)
README.md
```

## Sections (Homepage)

1. **Hero** — Name, tagline, key stats (GPA, internships, service hours)
2. **About** — Background and professional direction
3. **Experience** — GE Vernova, Data Nova, Live Long & Profit
4. **Skills** — Animated bars across data/analytics, product, and professional
5. **Leadership** — Student Senate, Beta Alpha Psi, Business Masterminds, Hindu Yuva
6. **Contact** — Email, LinkedIn, phone

## Tech

- Vanilla HTML / CSS / JavaScript (no frameworks, no dependencies)
- Google Fonts: DM Serif Display, DM Sans, Syne Mono
- IntersectionObserver API for scroll-triggered animations
- CSS custom properties for theming
- GitHub Pages for hosting

## Deployment Steps

1. Create a new GitHub repo (e.g. `portfolio`)
2. Push `index.html`, `about.html`, and `README.md` to the `main` branch
3. Go to **Settings → Pages → Branch: main → / (root) → Save**
4. Your live URL: `https://YOUR-USERNAME.github.io/portfolio/`
5. Update the Live Site link in this README
